# AWS EC2 Nginx Web Server Deployment

## Overview

This project demonstrates how to deploy a production-ready web server on AWS EC2 using Ubuntu and Nginx.

The server hosts a responsive website and is accessible through a public IP address over HTTP.

---

## Project Goal

- Launch an EC2 instance
- Configure secure SSH access
- Install and configure Nginx
- Deploy a static website
- Make the server publicly accessible

---

## Tech Stack

- AWS EC2
- Ubuntu Linux
- Nginx
- SSH
- SCP
- Windows PowerShell
- HTML
- CSS
- JavaScript
